# esu_n_esu
画像付きポスト投稿SNSアプリ。  
Flutter × Firebaseの勉強用に制作。  

## 使用技術
- 言語
  - Dart(Flutter)
- 状態管理
  - Provider
- 画像
  - image_picker
  - image_cropper
  - flutter_image_compress
- Fireabse
  - Firestore Database
  - Storage
  - Authentication

## 主な機能
- ポスト一覧表示
- ポストの新規投稿/編集/削除
- 画像アップロード
- ブックマーク
- ユーザーフォロー
- ログイン
- マイページ編集
